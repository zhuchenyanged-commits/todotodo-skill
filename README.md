# 待办事项清单 Skill

一个简单实用的命令行待办事项管理工具。

## 功能特性

### v2.0 新功能
- 🔴 任务优先级（高/中/低）
- 📅 任务到期日期
- 🎯 按优先级智能排序
- ✨ 可视化优先级图标

### 基础功能
- ✅ 添加任务
- 📋 查看任务列表
- ✓ 标记任务完成
- 🗑️ 删除任务

## 安装要求

- Python 3.6+

## 使用方法

```bash
python3 todo.py
```

## 使用示例

```
==================================================
              待办事项清单 v1.0
==================================================

请选择操作:
1. ➕ 添加任务
2. 📋 查看任务
3. ✓ 完成任务
4. 🗑️  删除任务
0. 🚪 退出

请输入选项 (0-4): 1
请输入任务: 学习 Git Flow
✅ 已添加任务: 学习 Git Flow
```
